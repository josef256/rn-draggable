# rn-draggable

a react native wrapper to drag any component

## Installation

```sh
npm install rn-draggable
```

## Usage

```jsx
import { Draggable } from 'rn-draggable';

// ...

<Draggable>
	<YourCompomemt />
</Draggable>;
```
